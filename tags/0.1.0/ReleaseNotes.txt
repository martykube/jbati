
Release Notes - jBati - JavaScript ORM on Jaxer.

********************************************************************************
V0.1.0  2008-04-20

Initial Release:
This release provides the basic needed functionality to make jBati useful.

- Supports MySql, SqlLite not tested
- Client API includes: queryForObject, queryForList, insert, update, and remove
- sqlMapConfig - Supports use of the Jaxer appication DB connection 
	from local_jaxer/config.js
- sqlMap - supports specification of select, insert, update, and delete statements.
- sqlMap - supports replacement tokens for named object attributes and scalar
	parameters.
